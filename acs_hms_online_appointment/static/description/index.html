Detailed Description of the Module

Online Appointment Booking By ACS
Patients can book appointments by selecting an available date and time slot.
After choosing a date and slot, patients can make a payment and complete the booking.
Administrators can create slots manually or automatically.
Booking limits per slot can be configured as needed.
Slot creation is simplified using a wizard interface.
Online booked appointments are displayed in a separate menu for easy management.
Administrators can configure the number of days in advance appointments can be booked.
